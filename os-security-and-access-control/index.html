脚本：瀬古浩司

演出：荒井和人、砂小原巧

关于本集的制作思路, 演出 荒井和人
动画做得好真的归功动画公司吗?——关于动画公司拟人化的误区】 【精准空降到 03:01
咒术ep37演出分析 (bangumi.tv)

从开头将人行道标识和虎仗奔跑的身影重合，我就知道这集不简单，演出非常有想法。这集的构图、机位、不同镜头的使用、阴影、甚至画幅都玩出了花样。让标示牌服务于演出是本集的一大亮点。一切这些工作都是为了烘托一种怪异感，为了给最后“不存在的回忆”做铺垫。然而还是有两点不满：解说太多，太影响节奏；打戏每次都用一个套路，先是慢镜头再切镜头。一次两次观感还行，用多了真的审美疲劳。

历经持续的失望，就是令人振奋的ep37。
开场写实空间下的平行蒙太奇，使得见面时的升格极具魄力。
地铁站内的光源做的很好，赤血覆盖在灯上，对日常的破坏直接体现。
用灯牌解说演出很有创意，但有些脱靶了。。
厕所打戏部分，狭隘空间刁钻机位，“暴雨”与鲜血的绽放交相辉映，红与蓝的光 照射在角色身上雕刻出冷漠的锋芒，演出也平衡了双方的表现。“雨声”的反复用以表现角色内心有意思。
胀相“不存在的回忆”中刻意模糊的影像，隐隐的噪声，混乱的切镜时机和方向。以打碎玻璃平滑的脱出感情表达出彩。

除了原作内容（指解说）已经拉满的一集了。17:27到37秒里虎杖被踢墙落地时，撑手（侧墙）跳开攻击，落地站立迅速防御后互相对攻至双方互相抛摔至地（虎杖与胀湘优劣达到一致），这一段分镜在狭窄空间中对肉搏运动变化走向连通和截取，大范围大角度的移动镜头使双方打斗方式彼此交错和高涨并走向最终一刻（当然不止这十秒）。
额，大家普遍都在夸Apart的演出，但其实整个涩谷篇大部分集数演出特点都是带着这种异化来进行的，实际上前面集数将这种异化、不安【虽然各有想法】一直执行的很彻底且牢固，这集风评能好我真觉得是打斗部分的功劳。

空间感十分明了、具体的打戏分镜，演出张弛有度，镜头语言一直在表达，大量图案和颜色暗示，场中过渡机械丸解说时借助广告牌的文字辅助和逃跑时的视觉错位也让人耳目一新，OST更是神级配合、花样十足的音色和调调，加之强烈的轮廓高光和摄影泛光使局部和整体色调都处于美轮美奂的幻想里一般，再次回到前五集令人惊叹的水平中。
修正的精度已经无敌了，后半竟然又到了狭窄的室内，在优雅又不失狂野的红蓝对比色中开启了白刃战，这已经不只是年度级别水平了，简直是教科书式的打戏单回。
woc太帅了！在被出其不意的击中肩膀和脸庞后，BGM过一层滤波只留下鼓组的sub bass，本来躁动不安、摇曳不定的电吉他随着逐渐的优势一转成高昂的小号去烘托，借着第一人称的转身飞踢和蓝色灯牌的被摧毁，迎来最终的宣告胜利的一拳，实在太流畅了太出色了！！！！！！！！！！
草！被骗了，还有反转！
不单单他人呆了，突然的4:3让我也一瞬间脑袋空白，继续拉镜利用断续的灯光压缩画幅，只留下框住整个人的灰色构图，我艹了！Follow PAN再现用做正式切成4:3的buildup！我艹了！最后竟然还回收此前只是一笔带过的电梯做画幅的收缩和本集结束，我还能说什么……宇宙级神回…
