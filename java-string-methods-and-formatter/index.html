String 方法分类及示例

字符串检查方法（String Inspection Methods）

这些方法用于检查字符串的特性或内容。

方法 描述
length 返回字符串的长度（字符数量）。
charAt 返回指定索引处的字符。
indexOf 返回字符或子串在字符串中的首次出现位置的索引。
lastIndexOf 返回字符或子串在字符串中的最后一次出现位置的索引。
isEmpty 如果字符串长度为零，返回 true。
isBlank 如果字符串长度为零或只包含空白字符，返回 true。（JDK 11 新增）

字符串比较方法（String Comparison Methods）

用于比较两个字符串的内容。

方法 描述
contentEquals 如果字符串的值与传入的 CharSequence 参数相等，返回 true。
equals 如果字符串的值与参数字符串相等，返回 true。
equalsIgnoreCase 忽略大小写比较两个字符串是否相等，返回 true。
contains 如果字符串包含指定的字符序列，返回 true。
startsWith 检查字符串是否以指定前缀开头，返回 true。
endsWith 检查字符串是否以指定后缀结尾，返回 true。

示例：

字符串清理方法（String Cleanup Methods）

这些方法对字符串进行清理或转换操作。

方法 描述
indent 在多行文本的每行前增加或减少缩进（JDK 15 引入）。
strip 去除字符串两端的空白字符（支持更多 Unicode 空白字符，比 trim 更强）。
stripLeading 去除字符串开头的空白字符。
stripTrailing 去除字符串末尾的空白字符。
toLowerCase 将字符串转换为小写。
toUpperCase 将字符串转换为大写。

示例：

字符串操作方法（String Manipulation Methods）

这些方法对字符串进行修改或生成新字符串。

方法 描述
concat 将一个字符串连接到另一个字符串末尾。
join 使用分隔符将多个字符串连接成一个字符串。
repeat 返回重复指定次数的新字符串。
replace 替换字符串中的字符或子串。
substring 截取字符串的子串。

各部分详解

Argument Index

  • 用于指定哪个参数将被格式化。
  • 格式：n$，其中 n 是从 1 开始的参数索引。
  • 默认：如果没有提供，按参数出现顺序依次使用。

Flags

  • 修改输出格式的标志。
  • 常用标志：
标志 作用
- 左对齐（默认右对齐）。
+ 在数字前始终显示正号或负号。
0 使用零填充数字，直到达到指定宽度。
, 显示数字时使用千位分隔符。
( 将负数括在括号中，而不是显示负号。

示例：

Width

  • 指定最小输出宽度。
  • 如果实际内容短于宽度，则用空格填充。

示例：

Precision

  • 针对浮点数：指定小数点后的数字位数。
  • 针对字符串：限制输出的最大字符数。

Conversion

  • 指定输出数据的类型。
转换符 类型 示例
%s 字符串 System.out.printf("%s", "Hi");
%d 十进制整数 System.out.printf("%d", 10);
%f 浮点数 System.out.printf("%.2f", 3.14);
%x 十六进制整数 System.out.printf("%x", 255);

Formatter 与 String.format

Formatter 和 String.format 本质相同，区别在于：

  • String.format 返回格式化后的字符串。
  • Formatter 通常与文件或控制台输出结合。

总结

通过掌握 Formatter 的规则，开发者可以实现复杂的字符串格式化功能，无论是对齐、填充还是小数精度，都能精准控制输出效果。这种工具特别适合用于日志、报告和用户界面开发！
